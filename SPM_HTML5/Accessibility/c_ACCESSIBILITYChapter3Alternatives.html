Accessible alternatives for desktop

A number of user interface elements in the IBM Cúram Social Program Management (SPM) application have alternative accessibility features for keyboard and screen reader users on desktop computers.

  • The Calendar
    View calender events using the accessible Calendar List View.
  • The Cúram Express Rule Editor
    Create and publish new rule sets and modify existing ones in an accessible manner via the administration application.
  • The Data Store Editor
    You can create and modify data store schemas in an accessible manner.
  • The Intelligent Evidence Gathering Editor
    IEG scripts can be created and modified offline in an accessible manner.
  • Context Panel
    Use context panel toggle button to collapse the context panel to more easily access the navigation bar tabs.
  • Planning workspace
    Use the planning workspace to manage Activities and Objectives.
  • Pods
    Pods are user interface widgets that users can customize.
