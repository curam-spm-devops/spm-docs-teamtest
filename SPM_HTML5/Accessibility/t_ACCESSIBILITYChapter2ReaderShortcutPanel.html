Section shortcut panel

You can navigate and interact with the section shortcut panel in the IBM Cúram Social Program Management application by using a screen reader and keyboard.

The section shortcut panel contains links to allow quick access to common user areas within the IBM Cúram Social Program Management application. These links are grouped in section shortcut categories. Accessibility access features are integrated into the section shortcut panel to help keyboard and screen reader users.

Use the following steps to navigate and interact with the section shortcut panel by using a screen reader and keyboard:

  1. Navigate to the expand icon on the section shortcut panel using the TAB key and press ENTER. This expands the section shortcut panel.
  2. Press the Tab key to give focus to the expanded section shortcut category. JAWS identifies this element as a Tab.
  3. Use the Up Arrow and Down Arrow keys to navigate through each section shortcut category in the section shortcut panel. The section is automatically expanded when it receives focus.
  4. When a section shortcut category receives focus, you can press the Tab key to access each link in that category.
