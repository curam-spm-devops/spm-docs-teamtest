Starting the WebLogic Administration Service

  1. Open a command prompt and navigate to <WebLogic Install directory>/<domain_repository>/<DomainName> directory, (e.g. C:/Weblogic/user_projects/domains/<domain_name>). Execute the command startWeblogic from this directory. When the server has started correctly the following line should be displayed: <Server started in RUNNING mode> in the log file, which was configured previously in Java Virtual Machine Configuration or in the command prompt if it's not already configured.
  2. To open the Administration Console, the following URL should be opened in a web browser:

    http://<IP Address>:7001/console

  3. Login with the username and password configured during installation (e.g. weblogic / weblogic1).
