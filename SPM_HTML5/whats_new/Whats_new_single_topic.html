What's new in IBM Cúram Social Program Management

Read about the new features and modules, and updates to existing modules, in IBM Cúram Social Program Management.

The IBM Cúram Social Program Management 8.0.2 IBM documentation edition contains documentation for IBM Cúram Social Program Management 8.0.0, 8.0.1, and 8.0.2. Features and updates that are included for 8.0.1 and 8.0.2 are flagged, as shown in the following examples:
  • If an entire topic applies only from 8.0.2, an icon icon is displayed at the top of the topic. The icon is also displayed next to the topic where it is displayed within the list of child topic links at the end of the parent topic.
  • If part of a topic applies from 8.0.2, an icon icon is displayed at the start of content that applies only from 8.0.2.

    The content is also highlighted in light blue, to indicate where the 8.0.2 content starts and ends, as shown here.
